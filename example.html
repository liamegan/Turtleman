This example demonstrates the Turtleman library for creating SVG graphics using turtle graphics programming.

Code Example:

// Create a new turtle instance
const turtle = new TurtleSVG({
    width: 600,
    height: 400,
    strokeColour: "blue",
    strokeWidth: 2
});

// Draw a square
for (let i = 0; i < 4; i++) {
    turtle.forward(100);
    turtle.right(90);
}
